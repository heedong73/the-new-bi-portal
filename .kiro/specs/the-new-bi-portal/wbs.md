# WBS — The New BI Portal (BIP)

WBS 작성 참고용. `tasks.md`(작업 상세) / `requirements.md`(R번호) / `design.md`(설계) / `risk & decision_log.md`(D번호)에서 도출한 **기능 단위 작업 분해**이다.

- 레벨 1 = 작업 패키지(WP, 정수 번호), 레벨 2 = 하위 작업(x.y). 레벨 3(액티비티)은 `tasks.md`의 각 Task 하위 불릿 참조.
- 상태: `완료` / `일부` / `예정`. 미완은 11.4(운영 이관)와 13(v1.1+)뿐.
- "관련 Task"는 `tasks.md`의 task 번호, "요구사항"은 `requirements.md`의 R 번호.
